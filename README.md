## <a name="introduction"></a>ü©∞ Introduction (English)

**Dance House** is a comprehensive software solution designed to modernize the operations of a premium ballet academy. It seamlessly integrates a brand website, an e-commerce boutique, and a robust CRM system into one unified platform.

This project streamlines administrative tasks, enhances student engagement, and drives business growth through automated tools and detailed analytics.

### ‚ú® Key Features

*   **Admin CRM**: Real-time dashboard analytics, student profile management, and membership tracking.
*   **Operations Excellence**: Smart class scheduling, attendance tracking, and instructor management.
*   **E-Commerce Boutique**: Full-featured online store for dancewear with integrated shipping options and VietQR payment.
*   **Student Portal**: Personal dashboard for students to view schedules and track membership status.
*   **Marketing Automation**: Lead management pipeline (Trial bookings) and automated expiration reminders.

### üöÄ Getting Started

